# Project 1: Excel Sales Dashboard 📊

## Objective
Excel ကိုသုံးပြီး Interactive Sales Dashboard တစ်ခုဆောက်ပါ။

## Dataset
`../../datasets/raw/sales_2025.csv` — 500 rows of sales data

## Dashboard Components
```
Page 1: Executive Summary
├── KPI Cards: Total Sales, Orders, AOV, Growth %
├── Line Chart: Monthly Sales Trend
├── Bar Chart: Top 10 Products
├── Pie Chart: Category Distribution
├── Map: Regional Performance
└── Slicers: Region, Category, Date Range
```

## Step-by-Step
1. Import sales_2025.csv into Excel
2. Convert to Table (Ctrl + T)
3. Create Pivot Tables for each analysis
4. Build charts from Pivot data
5. Add Slicers connected to all Pivot Tables
6. Design layout with company colors
7. Add conditional formatting for KPIs
