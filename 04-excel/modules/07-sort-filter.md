# Module 07: Sorting & Filtering

## 📌 ဒီ Module မှာလေ့လာရမည့်အကြောင်းအရာများ
- Sort (စီခြင်း)
- Filter (စစ်ထုတ်ခြင်း)
- Advanced Filter
- Custom Sort

## Sort (စီခြင်း)
Sort ဆိုတာ Data ကိုစနစ်တကျစီရတာပါ။

1. Data Tab → Sort
2. Column ရွေးပါ (e.g., Salary)
3. Order: Smallest to Largest / Largest to Smallest

### Custom Sort (Multiple Columns)
```
Sort by: Department (A to Z)
Then by: Salary (Largest to Smallest)
→ Department အလိုက်စီပြီး လစာအမြင့်ဆုံးကအနိမ့်ဆုံး
```

## Filter (စစ်ထုတ်ခြင်း)
Filter ဆိုတာ Data ထဲကသတ်မှတ်ချက်နဲ့ကိုက်တာကိုပဲပြဖို့ပါ။

1. Data Tab → Filter (or Ctrl + Shift + L)
2. Column Header မှာ ▼ ကိုနှိပ်ပါ
3. ပြချင်တဲ့အရာကိုရွေးပါ

### Number Filters
```
Greater Than 2000
Between 1000 and 3000
Top 10
Above Average
```

## Advanced Filter
1. Criteria Range ဆောက်ပါ
2. Data → Advanced
3. Criteria Range ကိုရွေးပါ
4. OK

## ✅ Key Takeaways
- Sort က A→Z (ascending) / Z→A (descending)
- Filter က data တွေကို hide လုပ်ပြီးစစ်ထုတ်တာ
- Multiple columns sort ကို Custom Sort နဲ့လုပ်ပါ
- Ctrl + Shift + L က Filter shortcut
