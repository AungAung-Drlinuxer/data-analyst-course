# Module 14: Slicer & Filter in Pivot Tables

## Slicer ဆိုတာဘာလဲ?
Visual filter button တွေပါ — Pivot Table ကို တစ်ချက်နှိပ်ရုံနဲ့ filter လုပ်လို့ရ။

## Slicer ထည့်နည်း
1. Pivot Table ထဲမှာ Click
2. Insert → Slicer
3. Field(s) ရွေးပါ → OK
4. Slicer ကိုနှိပ်ပြီး Filter လုပ်ပါ

### Slicer Features
- Multi-select: Ctrl + Click
- Clear filter: Eraser icon
- Connect to multiple Pivot Tables
- Custom colors & styles

## Timeline (Date Slicer)
1. Pivot Table → Insert → Timeline
2. Date field ကိုရွေးပါ
3. Months, Quarters, Years နဲ့ filter

## Pivot Table Filters vs Slicer
```
Filters: Drop-down menu, single selection
Slicer: Visual buttons, multi-select, better UX
```

## ✅ Key Takeaways
- Slicer က Pivot Table အတွက်အကောင်းဆုံး filter tool
- Timeline က date filter အတွက်သုံး
- Slicer တစ်ခုက Pivot Table အများကြီးကိုချိတ်လို့ရ
