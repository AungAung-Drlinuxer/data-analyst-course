## 🤔 Excel ဆိုတာဘာလဲ?

**Microsoft Excel** သည် **Spreadsheet Application** တစ်ခုဖြစ်ပြီး:
- Data တွေကို Row (အတန်း) နဲ့ Column (ဒေါင်လိုက်) ပုံစံနဲ့ စီစဉ်သိမ်းဆည်းနိုင်
- Formula နဲ့ Function တွေသုံးပြီး တွက်ချက်မှုများ ပြုလုပ်နိုင်
- Chart နဲ့ Graph ဆွဲပြီး Data Visualization လုပ်နိုင်
- Pivot Table နဲ့ Data Analysis လုပ်နိုင်
- VBA (Visual Basic for Applications) နဲ့ Automation လုပ်နိုင်

### Key Terms
| Term | အဓိပ္ပာယ် |
|------|-----------|
| **Workbook** | Excel ဖိုင် (.xlsx) — Sheets များစွာပါဝင် |
| **Worksheet** | Tab တစ်ခုချင်းစီ (Sheet) |
| **Cell** | Data ထည့်သွင်းရာနေရာ (e.g., A1, B2) |
| **Row** | အလျားလိုက်အတန်း (1, 2, 3...) |
| **Column** | ဒေါင်လိုက်အတန်း (A, B, C...) |
| **Range** | Cell အုပ်စု (e.g., A1:C10) |
| **Active Cell** | လက်ရှိရွေးချယ်ထားသော Cell |
