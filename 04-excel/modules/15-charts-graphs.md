# Module 15: Charts & Graphs in Excel

## 📌 Chart Types

| Chart | သုံးသင့်တဲ့အချိန် |
|-------|-------------------|
| **Column/Bar** | Category တွေကိုနှိုင်းယှဉ်ဖို့ |
| **Line** | Trend / Time Series ပြဖို့ |
| **Pie/Donut** | Proportion / Percentage ပြဖို့ |
| **Combo** | Mix of bar + line (different scales) |
| **Scatter** | Correlation ပြဖို့ |
| **Area** | Volume trend ပြဖို့ |
| **Waterfall** | အဆင့်ဆင့်ပြောင်းလဲမှုပြဖို့ |

## Chart ဆွဲနည်း
1. Data ကိုရွေးပါ
2. Insert → Chart Type ရွေးပါ
3. Design → Chart Style ပြောင်းပါ

## Chart Elements
```
Chart Title
Axis Titles
Legend
Data Labels
Data Table
Error Bars
Trendline
```

## Recommended Charts
Excel 2013+ မှာ "Recommended Charts" ဆိုတဲ့ feature ပါတယ်။
Data ကိုရွေးပါ → Insert → Recommended Charts
Excel က သုံးသင့်တဲ့ Chart ကို အလိုအလျောက်ရွေးပေးမယ်။

## ✅ Key Takeaways
- Chart type က data ရဲ့သဘောသဘာဝပေါ်မူတည်ပြီးရွေးပါ
- Line Chart → Trends, Bar Chart → Comparison, Pie → Composition
- Chart Elements ကို customize လုပ်နိုင်
- Recommended Charts က beginner တွေအတွက်အထောက်အကူဖြစ်
