# Module 16: Data Visualization Course

## 📌 Visualization Best Practices

### 1. Less is More
```
❌ Too many colors, 3D effects, cluttered labels
✓ Clean, simple, focused on data
```

### 2. Choose Right Chart
```
Comparison → Bar/Column Chart
Trend → Line Chart
Composition → Pie/Stacked Bar
Relationship → Scatter Plot
Distribution → Histogram
```

### 3. Color Usage
- Max 5-7 colors per dashboard
- Use company brand colors
- Red = Warning/Negative, Green = Positive
- Color-blind friendly palette

### 4. Dashboard Layout
```
Title Bar (Bold, Center)
├── KPI Row (4-6 Key Metrics)
├── Main Chart (Largest)
├── Supporting Charts (2-3)
└── Slicers / Filters (Top/Side)
```

## Excel Dashboard Example
```
Sales Dashboard:
├── KPI: Total Sales, Orders, Customers, AOV
├── Line Chart: Sales Trend (Monthly)
├── Bar: Top Products
├── Pie: Category Distribution
└── Slicers: Region, Date
```

## ✅ Key Takeaways
- Keep it simple — data ကအဓိက
- Right chart type ကိုရွေးပါ
- Color တွေကို purpose နဲ့သုံးပါ
- Dashboard layout က logical flow ရှိရမယ်
