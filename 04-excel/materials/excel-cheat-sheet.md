# Excel Cheat Sheet 🚀

## Navigation Shortcuts
| Key | Action |
|-----|--------|
| Ctrl + Arrow | Jump to edge |
| Ctrl + Home | Cell A1 |
| Ctrl + End | Last cell |
| Ctrl + Page Up/Down | Switch sheets |
| F5 | Go To |

## Selection Shortcuts
| Key | Action |
|-----|--------|
| Shift + Arrow | Extend selection |
| Ctrl + Shift + End | Select to last |
| Ctrl + A | Select all |
| Ctrl + Space | Select column |
| Shift + Space | Select row |

## Formatting Shortcuts
| Key | Action |
|-----|--------|
| Ctrl + 1 | Format cells |
| Ctrl + B | Bold |
| Ctrl + I | Italic |
| Ctrl + Shift + $ | Currency format |
| Ctrl + Shift + % | Percentage format |

## Formula Shortcuts
| Key | Action |
|-----|--------|
| = | Start formula |
| Alt + = | AutoSum |
| F2 | Edit cell |
| F4 | Toggle absolute/relative reference |
| F9 | Calculate |
| Ctrl + ` | Show formulas |
