# 🍕 Pizza Sales Dashboard (Food Service)

## Domain: Food Service

Pizza Sales Data ကိုခွဲခြမ်းစိတ်ဖြာပြီး Popular Items, Peak Hours, Category Performance များကို Dashboard ဖြင့်ပြသခြင်း။

## Objectives
- Best/Worst Selling Pizzas
- Peak Order Times
- Category-wise Revenue
- Daily/Weekly Sales Trends

## Dataset
Pizza Sales Data (Kaggle)

## Key Features
- Bar Charts for Popular Items
- Line Charts for Hourly Trends
- Matrix for Day × Category

## Deliverables
- Sales Performance Dashboard
- Menu Analysis Report
- Time-based Trends Analysis
