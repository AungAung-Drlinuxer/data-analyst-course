# 🚦 Road Accident Report (Transport & Safety)

## Domain: Transport

Road Accident Data ကိုခွဲခြမ်းစိတ်ဖြာပြီး Accident Hotspots, Severity Analysis, Time Patterns များကို Dashboard ဖြင့်ပြသခြင်း။

## Objectives
- Accident Severity Analysis
- High-Risk Locations (Hotspots)
- Time of Day Patterns
- Vehicle Type Involvement

## Dataset
UK Road Accidents (Kaggle)

## Key Features
- Map Visual with Heatmap
- Pie/Donut for Severity Distribution
- Slicers: Year, Weather, Road Type

## Deliverables
- Safety Dashboard
- Accident Hotspot Map
- Trend Analysis Report
