# 🏏 IPL Analysis (Sports Analytics)

## Domain: Sports Analytics

IPL Cricket Data 2008-2024 ကိုခွဲခြမ်းစိတ်ဖြာပြီး Team Performance, Player Stats, Match History များကို Dashboard ဖြင့်ပြသခြင်း။

## Objectives
- Team Win/Loss Analysis
- Player of the Match Trends
- Venue/Home Ground Advantage
- Season-wise Performance

## Dataset
IPL Dataset (Kaggle/ESPNcricinfo)

## Key Features
- Parameter for Season Selection
- Drill-through: Season → Match
- Custom Visuals for Cricket Stats

## Deliverables
- IPL Dashboard (All Seasons)
- Team Comparison Report
- Player Performance Analysis
