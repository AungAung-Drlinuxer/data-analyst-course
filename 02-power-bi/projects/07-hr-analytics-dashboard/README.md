# 👥 HR Analytics Dashboard (Human Resources)

## Domain: HR

Employee Data ကိုခွဲခြမ်းစိတ်ဖြာပြီး Attrition, Diversity, Performance, Headcount များကို Dashboard ဖြင့်ပြသခြင်း။

## Objectives
- Attrition Rate & Reasons
- Diversity & Inclusion Metrics
- Department-wise Headcount
- Performance Rating Distribution

## Dataset
HR Employee Attrition Data

## Key Features
- AI Key Influencers for Attrition
- Decomposition Tree for Root Cause
- RLS for Department Managers

## Deliverables
- HR Executive Dashboard
- Attrition Analysis Report
- Diversity Scorecard
