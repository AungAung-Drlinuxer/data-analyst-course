# 📦 Ecommerce Sales Analysis (Supply Chain)

## Domain: Supply Chain

E-commerce Supply Chain Data ကိုခွဲခြမ်းစိတ်ဖြာပြီး Order Fulfillment, Shipping, Returns များကို Dashboard ဖြင့်ပြသခြင်း။

## Objectives
- Order Fulfillment Time
- Shipping Cost Analysis
- Return Rate by Product
- Warehouse Performance

## Dataset
E-commerce Supply Chain Data

## Key Features
- Funnel Chart for Order Process
- Map for Shipping Routes
- Slicers: Warehouse, Product Category

## Deliverables
- Supply Chain Dashboard
- Logistics Performance Report
- Returns Analysis
