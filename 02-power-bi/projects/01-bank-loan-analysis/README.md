# 🏦 Bank Loan Analysis (Finance)

## Domain: Finance

Bank Loan Data ကိုခွဲခြမ်းစိတ်ဖြာပြီး Loan Distribution, Default Rate, Customer Demographics များကို Dashboard ဖြင့်ပြသခြင်း။

## Objectives
- Loan Approval Rate Analysis
- Default Rate by Customer Segment
- Loan Amount Distribution by Region
- Monthly Loan Trends

## Dataset
Bank Loan Data (Kaggle)

## Key Features
- DAX Measures: Loan Approval Rate, NPA %
- Visuals: Waterfall, Pie, Line Charts
- Slicers: Region, Loan Type, Date

## Deliverables
- Interactive Dashboard with KPIs
- Executive Summary Page
- Loan Performance Report
