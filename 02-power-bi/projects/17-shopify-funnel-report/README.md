# 🛍️ Shopify Funnel Report (Digital Commerce)

## Domain: Digital Commerce

Shopify Store Data ကိုခွဲခြမ်းစိတ်ဖြာပြီး Conversion Funnel, Cart Abandonment, AOV များကို Dashboard ဖြင့်ပြသခြင်း။

## Objectives
- Conversion Rate by Stage
- Cart Abandonment Analysis
- Average Order Value (AOV)
- Customer Lifetime Value

## Dataset
Shopify E-commerce Data

## Key Features
- Funnel Chart for Conversion
- Cohort Analysis for Retention
- AI Key Influencers for Abandonment

## Deliverables
- E-commerce Funnel Dashboard
- Customer Journey Report
- AOV Analysis
