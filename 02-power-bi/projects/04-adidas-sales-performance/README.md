# 👟 Adidas Sales Performance (Retail)

## Domain: Retail

Adidas Sales Data ကိုခွဲခြမ်းစိတ်ဖြာပြီး Product Category, Region, Sales Channel များကို Dashboard ဖြင့်ပြသခြင်း။

## Objectives
- Sales by Product Category
- Regional Sales Performance
- Online vs Retail Sales
- Profit Margin Analysis

## Dataset
Adidas US Sales Dataset

## Key Features
- Map Visualization for Regions
- Waterfall Chart for Profit Analysis
- What-If Parameter for Discount

## Deliverables
- Regional Sales Dashboard
- Product Category Report
- Channel Performance Analysis
