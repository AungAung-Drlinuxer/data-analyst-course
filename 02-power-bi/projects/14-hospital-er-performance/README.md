# 🏥 Hospital ER Performance (Healthcare)

## Domain: Healthcare

Hospital ER Data ကိုခွဲခြမ်းစိတ်ဖြာပြီး Wait Time, Patient Flow, ER Utilization များကို Dashboard ဖြင့်ပြသခြင်း။

## Objectives
- Average Wait Time by Severity
- Patient Arrival Patterns
- ER Bed Utilization
- Department Referral Analysis

## Dataset
Hospital ER Data

## Key Features
- Gauge for Wait Time Targets
- Line Chart for Hourly Arrivals
- Drill-through: ER → Department

## Deliverables
- ER Performance Dashboard
- Patient Flow Analysis
- Resource Utilization Report
