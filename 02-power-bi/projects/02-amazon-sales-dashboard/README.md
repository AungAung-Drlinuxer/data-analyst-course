# 🛒 Amazon Sales Dashboard (E-Commerce)

## Domain: E-Commerce

Amazon Sales Data ကိုခွဲခြမ်းစိတ်ဖြာပြီး Product Performance, Sales Trends, Customer Behavior များကို Dashboard ဖြင့်ပြသခြင်း။

## Objectives
- Sales by Category & Sub-category
- Top Selling Products
- Customer Segmentation
- Profit Analysis by Region

## Dataset
Amazon Sales Data

## Key Features
- Multi-page Report
- Drill-through: Category → Product
- Hierarchies for Date & Geography

## Deliverables
- Sales Performance Dashboard
- Product Analysis Report
- Customer Insights Page
