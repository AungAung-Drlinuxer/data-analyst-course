# 🚗 Car Sales Dashboard (Automotive)

## Domain: Automotive

Car Sales Data ကိုခွဲခြမ်းစိတ်ဖြာပြီး Model Performance, Inventory Analysis, Sales Trends များကို Dashboard ဖြင့်ပြသခြင်း။

## Objectives
- Sales by Make & Model
- Inventory Turnover Analysis
- Price Trend Analysis
- Seasonal Sales Patterns

## Dataset
Car Sales Data (Kaggle)

## Key Features
- Matrix Visual for Brand × Model
- Gauge Charts for Targets
- Decomposition Tree for Root Cause

## Deliverables
- Sales Performance Dashboard
- Inventory Analysis Report
- Price Trend Analysis
