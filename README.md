## 📚 သင်ရိုးသုံးပိုင်း ခြုံငုံသုံးသပ်ချက်

### 1️⃣ SQL — ဒေတာဘေ့စ် အခြေခံအုတ်မြစ် (၂၂ ခန်း)

| အကြောင်းအရာ | ရှင်းလင်းချက် |
|---|---|
| **DDL Commands** | CREATE, ALTER, DROP, TRUNCATE — Database နှင့် Table များ တည်ဆောက်ခြင်း |
| **DQL Commands** | SELECT, WHERE, JOIN, Subquery — ဒေတာများ ရှာဖွေထုတ်ယူခြင်း |
| **Window Functions** | RANK, DENSE_RANK, ROW_NUMBER, LEAD/LAG — အဆင့်မြင့် ဒေတာတွက်ချက်ခြင်း |
| **CTEs** | Common Table Expressions — ရှုပ်ထွေးသော Query များကို စနစ်တကျရေးသားခြင်း |
| **Functions** | Text, Numeric, Date/Time, Control Flow — Function အမျိုးမျိုး |
| **Views** | Virtual Table များ ဖန်တီးအသုံးပြုခြင်း |
| **Projects** | လက်တွေ့ပရောဂျက်များ |

> 📌 **သင်ယူရမည့် အစီအစဉ်** — Module 1 → 2 → 3 → ... → 22 (တစ်ခန်းချင်းစီ အစဉ်လိုက်)

### 2️⃣ Power BI — Data Visualization နှင့် Business Intelligence (၂၈ ခန်း)

| အကြောင်းအရာ | ရှင်းလင်းချက် |
|---|---|
| **Fundamentals** | Power BI Desktop, Service, Mobile — အခြေခံသဘောတရား |
| **Data Transformation** | Power Query — ဒေတာသန့်ရှင်းရေးနှင့် ပြောင်းလဲခြင်း |
| **Data Modeling** | Relationships, Star Schema — ဒေတာပုံစံချခြင်း |
| **DAX** | Measures, Calculated Columns, Time Intelligence — တွက်ချက်မှုဆိုင်ရာ |
| **Visualization** | Charts, Slicers, Filters, Hierarchies — မြင်ကွင်းဖော်ပြချက်များ |
| **AI & Advanced** | Claude AI, AI Visuals, AutoML — ဉာဏ်ရည်တုအဆင့်မြင့်ခွဲခြမ်းစိတ်ဖြာခြင်း |
| **Projects ၂၀** | Finance, Sales, HR, Healthcare, Sports စသည့် နယ်ပယ်စုံ |

## Additional Course Information

### Prerequisites

ဤသင်တန်းကိုတက်ရောက်ရန်အတွက် အောက်ပါအခြေခံများရှိရန်လိုအပ်ပါသည်-

1. **Basic Computer Skills** — File management, software installation
2. **Basic Data Concepts** — What is data, tables, records
3. **Analytical Thinking** — Problem-solving mindset
4. **Windows PC** — Power BI Desktop သည် Windows အတွက်သာ

### Tools Installation Guide

**Step 1: Install SQL Database**
- PostgreSQL: https://www.postgresql.org/download/
- Or SQL Server Dev: Free download from Microsoft

**Step 2: Install SQL GUI Tool**
- DBeaver (Recommended): https://dbeaver.io/
- SSMS: For SQL Server only

**Step 3: Install Power BI Desktop**
- Microsoft Store: Search 'Power BI Desktop'
- Or Download Center: Microsoft official site

**Step 4: Microsoft Fabric Access**
- Free trial: https://app.fabric.microsoft.com
- F64 capacity for 60 days

### How to Use This Course

1. **Start with SQL** — Database အခြေခံသည်အရေးကြီးဆုံးဖြစ်ပါသည်
2. **Practice Daily** — တစ်ရက်ကို အနည်းဆုံး 1-2 နာရီလေ့ကျင့်ပါ
3. **Do the Exercises** — Module တစ်ခုစီရှိ exercise များကိုလုပ်ဆောင်ပါ
4. **Build Projects** — Project များကိုကိုယ်တိုင်တည်ဆောက်ပါ
5. **Join Communities** — Online forums, Facebook groups, Discord
